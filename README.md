# Cairos Media Landing Page

A responsive, single-page website concept for Cairos Media with sections for Home, Services, Portfolio, About, and Contact.

## Preview locally

Open `index.html` directly in a browser, or run a simple local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Before publishing

1. Replace portfolio placeholders with Vimeo or YouTube embeds.
2. Connect the contact form to a CRM, form endpoint, or email service.
3. Add real portfolio thumbnails, project credits, and approved performance metrics.
4. Add Privacy Policy and Terms pages when forms and analytics go live.

## Files

- `index.html` — page structure and copy
- `styles.css` — visual system and responsive layout
- `script.js` — navigation, animations, modal previews, and form interaction
- `assets/favicon.svg` — Cairos icon
